# CucumberMe

CucumberMe is your way of anonymously sending cucumbers to a friend, ex or anyone you want! Show some personality with a clever message written on your cucumber.

## Built With 

* HTML5
* CSS3
* Vanilla JavaScript
* jQuery
* Shopify
* Love
